### 五大发现

1. **确定性方案在纯态+中等 N 下稳健更优** — qutrit N=16: ratio=0.915, 2Q N=72: ratio=0.964
2. **态类型是关键混淆因子** — 混合态优势(~3%) < 纯态(~4%)
3. **局域性是决定性内部因子** — 1-local 确定性始终最优，3-local 随机更优（shadow norm 3 vs 27）
4. **影子范数理论-实验吻合** — ||P||²_shadow = 3^k for k-local Pauli
5. **维度效应非单调** — d=3 (qutrit) ratio 0.91 最强，d=8 (3-qubit) ratio 0.98 最弱

## Part D: W2 — 定理形式化

### 定理 1 (MUB = DPRT 代数等价)

对素数 p 和 d+1 组 Wootters MUB，存在代数双射 Φ 将 MUB 投影测量集合映射到 DPRT 投影方向集合。DPRT 反演公式 `ρ = Σ pΠ - I` 的误差为机器零（Frobenius 范数 ≤ 10⁻¹⁵）。

### 定理 2 (方差上界)

对确定性遍历方案，估计方差满足：
```
E‖ρ̂ - ρ‖²_F ≤ d(d+1)² / N
```
上界保守（紧度 1.2-7.5×），但提供了 O(d³/N) 的渐近保证。

### 定理 3 (相变点)

- T3a (全态): ratio → 1，收敛 O(1/√d) ✓
- T3b (1-local): ratio 振荡不收敛，由 F_d 原根和乘法子群决定 ✓
- d=7 为最优值点，d=13 为次优点（原根=2），d>11 后收敛至 ~0.96
- T3c (k≥2): ratio → 1，收敛 O(1/3^k)（预测）
