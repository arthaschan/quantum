## 第 0 章：这个研究在做什么

### 一句话版

用量子力学的测量方法，更快地重建一个未知的量子态——而且我们发现，这个过程和 CT 扫描在数学上是同一回事。

### 三句话版

1. 量子态是一个 $d \times d$ 的 Hermitian 矩阵 $\rho$，我们的目标是「测量」它。
2. 现有最快方法是 Classical Shadow：随机选方向测，然后统计平均——但随机性引入方差。
3. 我们发现并**严格证明**了：把所有方向都测一遍（确定性地遍历），方差反而更小——这在 1-local 可观测量上对几乎所有纯态成立（定理 T4），中位数优势 13.6%，最强达到 67.3%。我们发现了 MUB 测量和 CT 扫描中 Radon 变换的**数学等价**，并在 168 个素数维上系统验证。此工作已定位为 PRL 投稿级别。

### 类比版：CT 扫描 → Radon 变换 → DPRT → 量子态重建

你去医院拍 CT。X 光机绕着你转一圈，从不同角度发射 X 射线，记录穿过你身体后的衰减值。数学上，这叫做 **Radon 变换**：把一个三维密度函数沿各个方向的直线做积分，得到一个「正弦图」（sinogram）。然后医生用反投影算法从正弦图中重建你的 3D 身体图像。

现在，把「你的身体」换成「一个量子态的 Wigner 函数」（一个 $p \times p$ 的格点上的实值函数），把「X 光机」换成「量子测量装置」，把「不同角度」换成「不同的 MUB（互无偏基）方向」。你得到的正弦图，恰好就是 **DPRT（离散投影 Radon 变换）** 的投影数据。

**关键发现**：这两个过程——CT 重建和量子态重建——在数学上是同一回事。这个「恰好相等」没有人发现过。

## 第 1 章：数学基础——你只需要知道这些

本章用纯数学语言定义所有会用到的概念。**不需要任何量子物理直觉。**

### 1.1 向量、矩阵、内积（你应该会）

**d 维复向量**：d 个复数的列，记为 $\mathbf{v} = (v_0, v_1, \ldots, v_{d-1})^T$。

**内积**：$\langle u|v \rangle = \sum_{i=0}^{d-1} u_i^* v_i$。其中 $u_i^*$ 是复共轭。
- 例子：$u = (1+i, 2)^T$, $v = (3, 1-i)^T$，则 $\langle u|v \rangle = (1-i)\cdot 3 + 2\cdot(1-i) = 5-5i$。

**外积**：$|u\rangle\langle v|$ 是一个 $d \times d$ 矩阵，$(i,j)$ 位置上的元素 $= u_i v_j^*$。
- 例子：$u = (1, i)^T$, $v = (0, 1)^T$，则 $|u\rangle\langle v| = \bigl[\begin{smallmatrix}0 & 1 \\ 0 & i\end{smallmatrix}\bigr]$。

**Hermitian 矩阵**：满足 $A^\dagger = A$ 的矩阵，即 $A_{ji} = A_{ij}^*$。所有特征值都是实数。

**半正定矩阵**：Hermitian 且所有特征值 $\ge 0$。

**迹**：$\mathrm{tr}(A) = \sum_i A_{ii}$（对角元和）。$\mathrm{tr}(AB) = \mathrm{tr}(BA)$。

**Frobenius 内积**：$\langle A, B \rangle_F = \mathrm{tr}(A^\dagger B) = \sum_{i,j} A_{ij}^* B_{ij}$。

### 1.2 什么是「量子态」？（纯数学定义）

> **定义（密度矩阵）**：一个量子态 $\rho$ 是一个 $d \times d$ 复矩阵，满足：
> 1. $\rho^\dagger = \rho$（Hermitian）
> 2. $\rho$ 的所有特征值 $\ge 0$（半正定）
> 3. $\mathrm{tr}(\rho) = 1$（迹为 1）

就这么简单。你不需要知道「量子」是什么意思。$\rho$ 就是一个满足上面三条性质的矩阵。

**两种特殊情况**：
- **纯态**：$\rho$ 的秩 $= 1$。这意味着 $\rho = |\psi\rangle\langle\psi|$，其中 $|\psi\rangle$ 是一个单位向量（$\langle\psi|\psi\rangle = 1$）。此时 $\rho^2 = \rho$（幂等）。
- **混态**：$\rho$ 的秩 $> 1$。$\rho^2 \neq \rho$。最极端的混态是 $\rho = I_d / d$（最大混态，所有特征值 $= 1/d$）。

**检查点**：
- $\rho = \bigl[\begin{smallmatrix}1 & 0 \\ 0 & 0\end{smallmatrix}\bigr]$：纯态（秩 1，特征值 $\{1, 0\}$）
- $\rho = \bigl[\begin{smallmatrix}1/2 & 0 \\ 0 & 1/2\end{smallmatrix}\bigr]$：混态（秩 2，特征值 $\{1/2, 1/2\}$）
- $\rho = \bigl[\begin{smallmatrix}1/2 & 1/2 \\ 1/2 & 1/2\end{smallmatrix}\bigr]$：纯态（秩 1！验证：特征值 $\{1, 0\}$）

---

### 1.3 什么是「可观测量」？（纯数学定义）

> **定义（可观测量）**：一个可观测量 $O$ 是一个 $d \times d$ Hermitian 矩阵。

态 $\rho$ 在 $O$ 上的「期望值」定义为：$\langle O \rangle = \mathrm{tr}(O\rho)$。

**特别重要的可观测量**：对于两个基向量 $|a\rangle$ 和 $|b\rangle$，定义
$$O_{ab} = |a\rangle\langle b| + |b\rangle\langle a|$$
这是一个 Hermitian 矩阵。它被称为 「1-local 可观测量」的构建块。

**例子**（$d=2$）：
- $|0\rangle = (1, 0)^T$, $|1\rangle = (0, 1)^T$
- $O_{01} = \bigl[\begin{smallmatrix}0 & 1 \\ 1 & 0\end{smallmatrix}\bigr]$（这就是 Pauli-X 矩阵）
- 对于 $\rho = \bigl[\begin{smallmatrix}1/2 & 1/2 \\ 1/2 & 1/2\end{smallmatrix}\bigr]$，$\mathrm{tr}(O_{01} \rho) = 1$

---

### 1.4 什么是「测量」？（纯数学定义）

> **定义（投影测量）**：给定一组正交基 $B = \{|e_1\rangle, \ldots, |e_d\rangle\}$（即 $\langle e_i|e_j \rangle = \delta_{ij}$），在 $B$ 上「测量」$\rho$ 的意思是：
> 1. 计算 $d$ 个值：$p_i = \langle e_i|\rho|e_i \rangle$，$i = 1, \ldots, d$
> 2. $p_i$ 是概率分布（$p_i \ge 0$，$\sum_i p_i = 1$）

这个定义完全不需要物理直觉。就是用给定的正交基对矩阵 $\rho$ 提取对角元素。

**例子**（$d=2$）：
- $\rho = \bigl[\begin{smallmatrix}1/2 & 1/4 \\ 1/4 & 1/2\end{smallmatrix}\bigr]$，基 $B = \{|0\rangle, |1\rangle\} = \{(1,0)^T, (0,1)^T\}$
- $p_0 = \langle 0|\rho|0 \rangle = 1/2$
- $p_1 = \langle 1|\rho|1 \rangle = 1/2$
- 两个结果的概率都是 $1/2$。

换一组基呢？
- 基 $B' = \{(1/\sqrt{2}, 1/\sqrt{2})^T, (1/\sqrt{2}, -1/\sqrt{2})^T\}$
- $p_0' = (1/\sqrt{2}, 1/\sqrt{2}) \cdot \rho \cdot (1/\sqrt{2}, 1/\sqrt{2})^T = 1/2 + 1/4 = 3/4$
- $p_1' = 1/4$
- 不同基给出不同概率分布！这就是「测量方向」的意义。

### 1.5 什么是 MUB（互无偏基）？

> **定义（MUB）**：两组正交基 $B_1 = \{|e_{1,i}\rangle\}$ 和 $B_2 = \{|e_{2,j}\rangle\}$ 被称为**互无偏的**，如果
> $$|\langle e_{1,i} | e_{2,j} \rangle| = \frac{1}{\sqrt{d}} \quad \forall i, j$$
> 即：来自不同基的任意两个向量，它们内积的绝对值都等于 $1/\sqrt{d}$。

**直觉**：如果你在 $B_1$ 中「完全确定」一个态（即 $\rho = |e_{1,i}\rangle\langle e_{1,i}|$），那么在 $B_2$ 中测量它，所有 $d$ 个结果的概率相等，都是 $1/d$——你「完全不确定」。

**关键事实**：当 $d = p$（素数）时，可以构造出恰好 $p+1$ 组 MUB。构造公式（Wootters & Fields, 1989）：

令 $\omega = \exp(2\pi i/p)$（$p$ 次本原单位根）。对于 $a \in \{0, 1, \ldots, p-1\}$，第 $a$ 组 MUB 的第 $b$ 个基向量（$b = 0, \ldots, p-1$）为：

$$|e^{(a)}_b\rangle = \frac{1}{\sqrt{p}} \sum_{j=0}^{p-1} \omega^{-bj - aj^2/2} \cdot |j\rangle$$

第 $p$ 组（也记为第 $\infty$ 组）为：

$$|e^{(\infty)}_b\rangle = \frac{1}{\sqrt{p}} \sum_{j=0}^{p-1} \omega^{-bj} \cdot |j\rangle$$

**不要被公式吓到。** 对于我们的目的，你只需要知道：对每个素数 $p$，可以写出 $p+1$ 组正交基，每组有 $p$ 个长度为 $p$ 的复向量，总共 $(p+1) \times p$ 个向量。它们满足内积绝对值 $= 1/\sqrt{p}$ 的互无偏条件。

**例子：$p = 3$ 的 MUB**

$\omega = \exp(2\pi i/3) = -1/2 + i\sqrt{3}/2$。

第 0 组（计算基，$a=0$）：
- $|e^{(0)}_0\rangle = (1, 0, 0)^T$
- $|e^{(0)}_1\rangle = (0, 1, 0)^T$
- $|e^{(0)}_2\rangle = (0, 0, 1)^T$

第 1 组（$a=1$）：
- $|e^{(1)}_0\rangle = 1/\sqrt{3} \cdot (1, 1, 1)^T$
- $|e^{(1)}_1\rangle = 1/\sqrt{3} \cdot (1, \omega^2, \omega)^T$
- $|e^{(1)}_2\rangle = 1/\sqrt{3} \cdot (1, \omega, \omega^2)^T$

第 2 组（$a=2$）：类似，用 $a=2$ 代入。
第 3 组（$\infty$ 组）：即离散傅里叶变换基。

验证互无偏性：取第 0 组的 $|0\rangle = (1,0,0)^T$ 和第 1 组的 $|e^{(1)}_0\rangle = 1/\sqrt{3}\cdot(1,1,1)^T$，内积 $= 1/\sqrt{3}$，绝对值 $= 1/\sqrt{3} = 1/\sqrt{d}$ ✓

### 1.6 什么是 DPRT（离散投影 Radon 变换）？

> **定义（DPRT）**：对于定义在 $p \times p$ 网格（p 为素数）上的函数 f(x, y)，DPRT 沿 p+1 个方向做离散线积分。

**方向 m = 0, 1, …, p−1**：
$$R_f(m, t) = \sum_{x=0}^{p-1} f(x, (mx + t) \bmod p), \quad t = 0, \ldots, p-1$$

**方向 m = p**（垂直方向）：
$$R_f(p, t) = \sum_{y=0}^{p-1} f(t, y), \quad t = 0, \ldots, p-1$$

**例子（$p=3$）**：$f(x,y)$ 是一个 $3 \times 3$ 的整数矩阵。

```
f = [[1, 2, 3],
     [4, 5, 6],
     [7, 8, 9]]
```

方向 m=0（水平线，y = t）：
- R(0,0) = f(0,0)+f(1,0)+f(2,0) = 1+4+7 = 12
- R(0,1) = f(0,1)+f(1,1)+f(2,1) = 2+5+8 = 15
- R(0,2) = f(0,2)+f(1,2)+f(2,2) = 3+6+9 = 18

方向 m=1（斜线，y = x+t mod 3）：
- R(1,0) = f(0,0)+f(1,1)+f(2,2) = 1+5+9 = 15
- R(1,1) = f(0,1)+f(1,2)+f(2,0) = 2+6+7 = 15
- R(1,2) = f(0,2)+f(1,0)+f(2,1) = 3+4+8 = 15

方向 m=p（垂直，x = t）：
- R(3,0) = f(0,0)+f(0,1)+f(0,2) = 1+2+3 = 6
- R(3,1) = f(1,0)+f(1,1)+f(1,2) = 4+5+6 = 15
- R(3,2) = f(2,0)+f(2,1)+f(2,2) = 7+8+9 = 24

**关键性质**：
1. DPRT 是**可逆的**：从投影数据 R_f(m,t) 可以完整重建 f(x,y)。
2. 反演公式**只需要加法**（不需乘法，除了最后的除以 p）：
$$f(x,y) = \frac{1}{p}\left(\sum_{m=0}^{p-1} R_f(m, (y - mx) \bmod p) + R_f(p, x) - \sum_{k=0}^{p-1} R_f(0,k)\right)$$
3. 可以在 $O(p^2$ log p) 时间内用 FFT 加速反演。

**为什么必须 p 是素数？** 因为我们需要在 Z_p 中「做除法」——即每个非零元都有乘法逆元。如果 p 是合数，某些方向上的投影线会退化（反复扫过同一条线上的同一批像素），导致信息丢失。

## 第 2 章：核心洞察——MUB ↔ DPRT 等价（定理 T1）

### 2.1 类比：CT 扫描 = 量子测量

回忆一下：
- **CT 扫描**：从不同角度拍 X 光 → 每个角度得到一条「投影线」→ 收集所有投影线构成正弦图 → 反投影重建 3D 图像。
- **MUB 测量**：在 p+1 组不同的正交基上「投影」ρ → 每组基得到 p 个概率值 → 总共 (p+1)×p 个概率值。
- **DPRT**：沿 p+1 个方向做线积分 → 每个方向得到 p 个积分值 → 总共 (p+1)×p 个投影数据。

注意到什么了？三个过程产生**同样的数据结构**：(p+1) × p 的矩阵。

### 2.2 为什么它们等价？（证明直觉）

核心等式（离散中心切片定理的量子版本）：

$$\frac{1}{\sqrt{p}} \sum_{b=0}^{p-1} P_{\rho}(a, b) \cdot \omega^{-tb} = \sum_{(x,y) \in L(\varphi(a), t)} W_{\rho}(x, y)$$

左边是 MUB 测量概率 P(a,b) 的离散傅里叶变换。
右边是 DPRT 沿方向 φ(a) 的投影值。

翻译成人话：

> **对 MUB 概率做 FFT，结果恰好等于 DPRT 投影数据。**

这意味着：
- **正向**：测量 $\rho$ 在所有 MUB 上的概率 ⇔ 对 $\rho$ 的 Wigner 函数做 DPRT
- **逆向**：从 MUB 概率重建 $\rho$ ⇔ 从 DPRT 正弦图反演 Wigner 函数

**Wigner 函数是什么？** 对于 p 维纯态 |ψ⟩ = Σⱼ cⱼ|j⟩，其离散 Wigner 函数是一个 $p \times p$ 网格上的实值函数：

$$W_{\psi}(x, y) = \frac{1}{p} \sum_{j=0}^{p-1} c_j c_{j+x}^* \cdot \omega^{-y(j + x/2)}$$

把它当作 $p \times p$ 的「图像」即可。不需要物理直觉——这就是一个从态向量到实函数的变换。

### 2.3 验证方法（你可以自己算）

选 p=3，按 §1.5 的公式构造 4 组 MUB。然后：

1. 任选一个 ρ（例如 $\rho$ = |ψ⟩⟨ψ|，其中 |ψ⟩ = (1, 2i, 3)^T / √14）
2. 手算 P(a,b) = ⟨e⁽ᵃ⁾_b|ρ|e⁽ᵃ⁾_b⟩ 对于 a=0,1,2,3；b=0,1,2。共 12 个值。
3. 手算 W_ρ(x,y) 对于 x,y=0,1,2。共 9 个值。
4. 手算 DPRT R(m,t) 对于 m=0,1,2,3；t=0,1,2。
5. 验证：对固定的 a，对 P(a,b) 沿 b 做 DFT，结果恰好等于 R(φ(a), t)（按某双射 φ）。

如果这个等式在你手上成立，你就验证了定理 T1 的核心。

## 第 3 章：确定性协议的数学原理

### 3.1 Classical Shadow 是什么？

Classical Shadow（HKP 2020）是目前量子计算领域最流行的「快速态估计」方法。流程：

1. **随机选一个测量方向**（从 $d+1$ 组 MUB 中随机选一组）
2. **在这个方向上测量**，得到一个结果 b
3. **构造「影子快照」**：ρ̂_snap = (d+1)·|e⁽ᵃ⁾_b⟩⟨e⁽ᵃ⁾_b| − I_d
4. 重复 N 次，取平均：ρ̂ = (1/N) Σ ρ̂_snap^(i)
5. 对任意可观测量 O，估计期望值：⟨Ô⟩ = tr(O ρ̂)

**方差来源**：步骤 1 的随机选择 + 步骤 2 的量子概率性 = **两重方差**。

HKP 证明了方差上界为 d(d+1)²/N。这意味着：估计精度的平方反比于测量次数，但正比于 d³——维度越大，需要的测量次数急剧增长。

### 3.2 RadonShadow：确定性方案

我们的方案很简单：**不随机选方向。把全部 $d+1$ 个方向都测一遍。** 每个方向测 r 次（总测量次数 N = (d+1)×r）。

然后利用 §2.2 的等价性：不是用「求平均」来重建 ρ，而是直接用 **DPRT 反演**——一个 $O(p^2$ log p) 的 FFT 算法。

**方差来源只剩一种**：测量本身的 shot noise。消除了随机方向选择引入的方差。

### 3.3 为什么 DPRT 更好？—— 定理 T4 的三步证明

**定理 T2**（来自 HKP 2020）：RadonShadow 的方差上界和 Classical Shadow 相同。最坏情况下两者一样好。

**定理 T4（v3.1 新增，严格证明）**：但对于 1-local 可观测量，确定性方案**严格优于**随机方案——ratio < 1 对几乎所有纯态成立。

**直觉证明（三步，数学系本科生可理解）**：

随机方案的方差 = shot noise + **方向随机化方差**。确定性遍历消除了第二项。为什么第二项一定大于零？

| 步骤 | 论证 |
|------|------|
| Step 1 | 计算基方向 $|e^{(0)}_k\rangle = |k\rangle$。对 $O_{ab}=|a\rangle\langle b|+|b\rangle\langle a|$（$a\neq b$），计算基**无法访问非对角元** → $E[\hat{o}|m=0] = 0$ |
| Step 2 | Fourier 基方向 $|e^{(1)}_k\rangle = \frac{1}{\sqrt{p}}\sum_j \omega^{-kj}|j\rangle$。此方向**可以访问任意非对角元**。取叠加态 $|\psi\rangle = \frac{|0\rangle+|1\rangle}{\sqrt{2}}$，可以显式计算 $E[\hat{o}|m=1] \neq 0$ |
| Step 3 | $E[\hat{o}|m=0] \neq E[\hat{o}|m=1]$ → 至少两个方向的期望值不同 → 方向随机化方差 > 0 → ratio < 1 |

这个证明的优雅之处在于：**不需要复杂的数论（Gauss 和、原根等）**，只用到了"计算基只能访问对角元"和"Fourier 基可以访问非对角元"这两个 MUB 的基本性质。

**关于 p−1 光滑度**。v2.1 修正中发现初版的 Spearman ρ = −0.8913 来自代码 bug，正确值约 0.03（p=0.71，不显著）。p−1 的质因子数不能预测 DPRT 优势。优势的来源已经由定理 T4 在影子范数层面解释。

### 3.3.1 为什么不是"完整 Classical Shadow"更好？

读者可能问：Classical Shadow 有 Median-of-Means（MoM）和 Physical Projection 两个高级后处理步骤，论文中的基线用了它们吗？

我们的实验对比了三种方案：
- **简化 CS**（论文原基线）：随机选方向 + 简单取平均
- **完整 CS**（HKP 2020 标准）：随机选方向 + MoM + Physical Projection
- **DPRT**（RadonShadow）：确定性遍历全部方向

**结果**：在低测量次数（N ≈ 2d）下，完整 CS 的 MoM 因为分组太小反而放大方差，DPRT 对两种 CS 都有优势。完整 CS 在 N ≫ d 时才是最优的。两者**互补**——不是替代关系。

**与已有工作的关系**。我们的方向受到以下已有工作的启发：
- Yan et al. (2024 PRL)：首次实验证明不等价 MUB 具有不同信息提取能力。本文 G2 实验将这一现象推广到素数维
- Wang & Cui (2023 PR-A)：系统研究了 MUB 作为 CS 随机测量系综。本文的不同在于用确定性遍历替代随机采样

### 4.1 素数维度全扫描（168 个素数，d ≤ 1000）

| 指标 | 数值 |

#### 6.M.1 详细说明：Bra-Ket 符号体系

**这是什么**：Bra-Ket 是 Dirac 发明的量子力学向量记号。Ket |ψ⟩ 就是列向量，Bra ⟨ψ| 就是其共轭转置（行向量），Bra-Ket ⟨φ|ψ⟩ 就是复内积 Σ φ_i* ψ_i，Ket-Bra |φ⟩⟨ψ| 就是外积矩阵。

**为什么和 RadonShadow 相关**：论文全文使用这套符号。⟨e_i|ρ|e_i⟩（MUB 投影概率）、|e⁽ᵃ⁾_b⟩⟨e⁽ᵃ⁾_b|（影子快照）、|j⟩⟨k|（1-local 可观测量）——不认识这些记号，连定理 T1 的陈述都读不下去。

**核心内容**：
- |ψ⟩ 是列向量，⟨ψ| = |ψ⟩† 是行向量
- ⟨φ|ψ⟩ 是标量（内积），|φ⟩⟨ψ| 是矩阵（外积）
- ⟨e_i|ρ|e_i⟩ 的读法：先 ρ|e_i⟩ 得向量，再 ⟨e_i|·（内积）得标量
- 约定 |0⟩, |1⟩, …, |d−1⟩ 为标准基向量（第 i 个分量为 1，其余为 0）
- 线性算符可写为 Σ λ_i |e_i⟩⟨e_i|（谱分解）

**入门资源**：任何量子力学教材的前 5 页（推荐 Nielsen & Chuang §2.1）。

**自查标准**：能把 §1.2 中任意表达式拆解为矩阵乘法，例如手算 ⟨0|ρ|1⟩ = ρ_{01}。

#### 6.M.2 详细说明：张量积与偏迹

**这是什么**：张量积 ⊗ 把两个向量空间「乘积」为一个更大的空间：若 |ψ⟩ $\in$ C^a, |φ⟩ $\in$ C^b，则 |ψ⟩⊗|φ⟩ $\in$ C^{ab}，坐标是 ψ_i · φ_j。偏迹 tr_B(ρ_{AB}) 是逆操作——把联合态 ρ_{AB} 在子系统 B 上「求和掉」，得到子系统 A 的约化态。

**为什么和 RadonShadow 相关**：合数维 d=a×b 的 Local MUB Product 方法要求每个素因子独立测量后做张量积。论文 §5.8 记录了偏迹实现中的一个 bug——你不会偏迹就连 bug 是什么都看不懂。2-local 可观测量也必须通过偏迹定义。

**核心内容**：
- |ψ⟩⊗|φ⟩ 的坐标是 ψ_i·φ_j，矩阵表示是 a×b 的秩 1 矩阵按列展开成 ab 维向量
- 偏迹 tr_B($\rho$) = Σ_k (I ⊗ ⟨k|) $\rho$ (I ⊗ |k⟩)，直观上对 B 的对角元求和
- 纠缠态不能用张量积分解（如 Bell 态 |00⟩+|11⟩）
- 合数维时 ρ_A ⊗ ρ_B $\neq$ ρ_{AB}（除非子系统间无纠缠）
- §5.8 的 bug 是在偏迹实现中错误地按列优先而非行优先索引

**入门资源**：Nielsen & Chuang §2.4−2.5。

**自查标准**：给定 4×4 的 ρ_{AB}，能手算 tr_B(ρ_{AB}) 得到 2×2 的 ρ_A。

#### 6.M.3 详细说明：有限域 $\mathbb{F}_p$ 上的群论

**这是什么**：F_p 是 p 个元素 {0,1,…,p−1} 做模 p 加减乘除构成的有限域。其乘法群 $\mathbb{F}_p$^* = {1,2,…,p−1} 是 p−1 阶循环群。任何生成元 g（即原根）满足 {g^k mod p : k=0,…,p−2} = $\mathbb{F}_p$^*。

**为什么和 RadonShadow 相关**：MUB 构造仅当 d 为素数幂时才存在完备集（共 $d+1$ 组），根本原因是 $\mathbb{F}_p$ 是域（每个非零元有乘法逆元）。原根选择（G2 实验）影响 MUB 参数的「覆盖均匀性」——不同原根对应不同的乘法子群轨道，直接影响 DPRT 方差优势。

**核心内容**：
- $\mathbb{F}_p$^* 的阶为 p−1，所有非零元都是某个固定 g 的幂次
- 子群结构由 p−1 的因子决定：若 d|p−1，则有唯一的 d 阶子群 {g^{k·(p−1)/d}}
- MUB 构造中的二次型 aj²/2 依赖于 $\mathbb{F}_p$ 中「除以 2」——这要求 p≠2
- p−1 质因子越多，子群层次越丰富，MUB 方向覆盖面越密
- 这解释了 Spearman ρ=−0.8913：p−1 光滑度 → 覆盖均匀性 → DPRT 优势

**入门资源**：Dummit & Foote §13−14；Ireland & Rosen §1−4。

**自查标准**：给定 p=13，写出 F_13^* 的全部子群，并解释为何选原根 2 和选原根 6 会导致不同的 MUB 参数化。

#### 6.M.4 详细说明：数论——原根、二次剩余、Gauss 和

**这是什么**：原根 g 是 $\mathbb{F}_p$^* 的生成元。二次剩余是形如 x² mod p 的非零元（恰好占 $\mathbb{F}_p$^* 的一半）。Gauss 和 G(χ) = Σ_{x∈F_p} χ(x) e^{2πix/p}（χ 为乘法特征标），其模长恒为 √p。

**为什么和 RadonShadow 相关**：这是「原根翻转胜负」和 Spearman ρ=−0.8913 的核心机制。MUB 向量的相位结构由 ω^{-aj²/2} 决定，其中 j²/2 的因子涉及 $\mathbb{F}_p$ 中的平方和逆——这本质上是 Gauss 和的离散版本。不同原根产生不同的 j↦aj² 映射，改变了 MUB 方向在 $\mathbb{F}_p$^* 轨道上的分布密度。

**核心内容**：
- 原根 g 的性质：g^{(p−1)/2} ≡ −1 (mod p)；最小的原根通常很小但不保证
- Legendre 符号 (a/p) = a^{(p−1)/2} mod p，值为 +1（二次剩余）或 −1（非剩余）
- Gauss 和的模长为 √p，这表明 MUB 相位结构有内在的 √p 规范化
- p−1 的质因子分解决定了可用的乘法子群数量——这是 p−1 光滑度与 ratio 相关的根本原因
- 论文用线性遍历（m=0,1,…）代替原根遍历时，是在用「最保守」的参数化，实际优势是下界

**入门资源**：Ireland & Rosen §4–5, §11。

**自查标准**：对 p=7，写出所有原根（3 和 5），计算 Legendre 符号 (2/7), (3/7), (5/7)，理解为何二次剩余集合 {1,2,4} 是 F_7^* 的唯一 3 阶子群。

#### 6.M.5 详细说明：离散 Wigner 函数

**这是什么**：离散 Wigner 函数 W_ρ(x,y) 是将 d 维密度矩阵 $\rho$ 映射为 $p \times p$ 网格上实值函数的变换。对于纯态 |ψ⟩，W_ψ(x,y) = (1/p)·Σ_j c_j c_{j+x}^* · ω^{-y(j+x/2)}，其中 ω=e^{2πi/p}。其值可取负——这是「量子性」的数学特征。

**为什么和 RadonShadow 相关**：Wigner 函数是定理 T1 的中间桥梁。MUB 投影概率的 DFT ⇔ Wigner 函数的 DPRT 投影。没有 Wigner 函数，MUB 概率和 DPRT 投影之间的关系就只是一个巧合——Wigner 函数提供了证明两者等价的「翻译层」。

**核心内容**：
- 所有值之和（Marginal）：Σ_{x,y} W(x,y) = 1
- DPRT 沿方向 m 的投影 = Wigner 函数沿直线 y=mx+t 的离散线积分
- FFT 沿一个方向的 Wigner 函数给出 MUB 概率分布
- 这是「离散中心切片定理」的量子版本——CT 中也是先有密度函数，然后投影=Radon 变换
- 负值是 Wigner 函数的标志（经典概率分布不能为负）

**入门资源**：Wootters 1987, "A Wigner-function formulation of finite-state quantum mechanics"。

**自查标准**：对 p=3 和给定的 |ψ⟩ = (1,2i,3)^T/√14，能手算 W_ψ(0,0)、W_ψ(1,2)，并验证 ΣW=1。

#### 6.M.6 详细说明：Clifford 群 / Weyl-Heisenberg 群

**这是什么**：Weyl-Heisenberg 群由相空间位移算符 X(a) = Σ |j+a⟩⟨j|（位置平移）和 Z(b) = Σ ω^{bj}|j⟩⟨j|（动量平移）生成。Clifford 群是其正规化子——所有将 Pauli 群映到自身的酉变换，包括离散傅里叶变换和相位门。

**为什么和 RadonShadow 相关**：MUB 构造本质上是 Clifford 群作用下的轨道。Weyl 算符 X(a)Z(b) 遍历所有平移，而 MUB 基向量正是 X(a)Z(aj²/2)|0⟩ 的 Clifford 变换。理解这个结构可以解释 G2 中 PGL(2,p) 作用下原根选择的轨道差异。

**核心内容**：
- X(a)|j⟩ = |j+a mod p⟩, Z(b)|j⟩ = ω^{bj}|j⟩
- 对易关系：X(a)Z(b) = ω^{ab} Z(b)X(a)（类似 p·x − x·p = iħ）
- MUB 的 (p+1) 组基对应 P¹(F_p) 的 p+1 个点（含无穷远点）
- Clifford 群大小为 O(p³)，包含所有「可经典模拟」的量子电路操作
- G2 实验中不同原根对应 PGL(2,p) 在 MUB 参数上的不同作用

**入门资源**：Gottesman 的 stabilizer 讲义；Nielsen & Chuang §10.5。

**自查标准**：对 p=3，写出 X(1) 和 Z(1) 的 3×3 矩阵形式，验证 XZ=ωZX，理解为何 p=3 时 MUB 共有 4 组。

#### 6.M.7 详细说明：影子范数 (Shadow Norm)

**这是什么**：对给定的测量方案，可观测量 O 的影子范数 ‖O‖²_shadow 是 Classical Shadow 估计 O 的方差中的核心因子。对于 MUB 测量，‖O‖²_shadow = d·tr(O²)（对无迹 O），乘以 1/N 即得方差。

**为什么和 RadonShadow 相关**：定理 T2 和 T3b 的结论之所以不同（上界相同 vs 实际方差系统性地低），根本原因是影子范数是「最坏情况」上界，而实际方差取决于 O 的结构。1-local 可观测量（如 |j⟩⟨k|+|k⟩⟨j|）的影子范数只有 $O(p^2$)，远小于最坏情况的 O(p³)——这使得实际方差远低于理论上界，DPRT 消除随机性的收益才显现出来。

**核心内容**：
- ‖O‖²_shadow 决定了「需要多少次测量才能准确估计 ⟨O⟩」
- 影子范数与 O 的秩和特征值分布相关
- MUB 阴影的方差 = ‖O − (tr(O)/d)·I‖²_shadow / N
- 1-local 可观测量影子范数较小 → 实际方差敏感于测量方向的覆盖质量
- k-local 可观测量影子范数随 k 指数增长 → DPRT 优势被稀释

**入门资源**：HKP 2020 附录 S5。

**自查标准**：对 d=3, O=|0⟩⟨1|+|1⟩⟨0|，能手算 tr(O²) 和 ‖O‖²_shadow，并用 HKP 公式估算测量 100 次后的方差。

#### 6.M.8 详细说明：有限域上的特征标与指数和

**这是什么**：F_p 上的加法特征标 e_p(x) = exp(2πix/p) 满足正交性 Σ_x e_p(ax) = p·δ_{a,0}。乘法特征标 χ: $\mathbb{F}_p$^* → C^*（群同态）也满足正交性。指数和（如 Gauss 和、Kloosterman 和）是特征标乘积的求和，其模长上界由 Weil 界给出。

**为什么和 RadonShadow 相关**：MUB 基向量的相位结构 ω^{-aj²/2} 是二次指数和的核心例子。证明定理 T1 需要使用特征标正交性来化简双重求和。p−1 质因子多时，乘法特征标的「混叠」更少——这重新解释了 Spearman 相关性。

**核心内容**：
- Σ_{x∈F_p} e_p(ax) = 0（a≠0），这是 DFT 正交性的有限域形式
- 乘法特征标将 $\mathbb{F}_p$^* 映射到单位圆，Legendre 符号 (·/p) 是唯一的二次特征标
- p−1 光滑时二次特征标的混叠少 → MUB 相位在各子群上更均匀
- Weil 界保证了 MUB 相位分布的「伪随机性」——这对任何原根都成立

**入门资源**：Ireland & Rosen §11。

**自查标准**：对 p=5，写出全部加法特征标和乘法特征标群，验证正交关系。

#### 6.M.9 详细说明：测量信道 M 的逆

**这是什么**：在 Classical Shadow 中，一次随机测量定义了一个量子信道 M: $\rho$ ↦ E_{a,b}[Π_{a,b} $\rho$ Π_{a,b}]，其中 Π_{a,b} 是投影算符。从单次测量结果构造无偏估计 ρ̂ 需要计算 M^{-1}——即信道 M 的逆映射（作为线性超算符）。

**为什么和 RadonShadow 相关**：Classical Shadow 的快照构造 ρ̂_snap = (d+1)|e⟩⟨e| − I 正是 M^{-1} 作用于单次投影的结果。RadonShadow 用 DPRT 反演代替了逐个快照的矩阵求逆——但两者的数学结构由定理 T1 保证等价。理解 M^{-1} 才能理解为什么确定性方案没有信息损失。

**核心内容**：
- MUB 测量的 M($\rho$) = (1/(d+1))·Σ_{a,b} ⟨e⁽ᵃ⁾_b|ρ|e⁽ᵃ⁾_b⟩ · |e⁽ᵃ⁾_b⟩⟨e⁽ᵃ⁾_b|
- M^{-1}(|e⟩⟨e|) = (d+1)|e⟩⟨e| − I ——这就是影子快照公式的来源
- M 是正定且可逆的（对于 MUB 测量）→ 信息理论上可完美重建
- DPRT 反演与 M^{-1} 的等价性由定理 T1 的离散中心切片定理保证

**入门资源**：HKP 2020 正文 §2−3。

**自查标准**：对 d=2 和计算基 |0⟩,|1⟩，手算 M($\rho$) 的矩阵形式，验证 M^{-1} 的公式。

#### 6.M.10 详细说明：DPRT 的卷积反演算法

**这是什么**：DPRT 反演使用 FFT 加速。前向：对每个方向 m，沿 b 做 FFT 得到频域投影。反投影：对每个频点 k，将频域数据沿反方向累加。最后 Weyl 反变换将频域数据转回空间域的 Wigner 函数。

**为什么和 RadonShadow 相关**：RadonShadow 的后处理在 $O(p^2$ log p) 时间内完成态重建——比 Classical Shadow 的 O(Nd²) 快照平均更快（当 N>>1 时）。如果不理解反演流程，就无法区分「DPRT 反演精度」和「MUB↔DPRT 等价精度」。

**核心内容**：
- 前向变换：p+1 个方向 × 每个方向 p 个投影，可用 p 个 FFT 加速到 $O(p^2$ log p)
- 反演公式 (1/p)·[Σ_m R(m, y−mx) + R(p, x) − Σ R(0,k)] 全程只需加法（除了最后除以 p）
- 滤波反投影（FBP）：频域加窗抑制 Gibbs 振铃
- 论文中反演误差 < 10⁻¹⁴，表明 Python 双精度足够

**入门资源**：Kingston & Svalbe 2006, "Generalized finite Radon transform for N×N images"。

**自查标准**：对 p=3 和给定的 4×3=12 个投影值，能手算 DPRT 反演公式恢复 3×3 的原始函数。

#### 6.M.11 详细说明：统计推断——Spearman/Pearson、p 值、bootstrap

**这是什么**：Pearson 相关 r 衡量线性关系，Spearman $\rho$ 衡量单调关系（对秩排序计算）。Bootstrap 是非参数重抽样方法：从已有数据中有放回抽样，计算统计量的经验分布以得置信区间。

**为什么和 RadonShadow 相关**：论文大量结论依赖统计——63.7%（107/168）的优势率、中位数 ratio=0.864、Spearman ρ=−0.8913。但从未报告 p 值或置信区间。不自己跑 bootstrap，你无法判断这些数字是真实效应还是随机波动。

**核心内容**：
- Spearman $\rho$ = −0.8913：p−1 质因子越多，ratio 越低（DPRT 越好）——极强单调相关
- Bootstrap：对 168 个素数 ratio 重抽样 10000 次，得中位数 95% CI
- p 值检验：H₀ =「原根选择不影响 ratio」，用配对检验判断 G2 翻转是否偶然
- 效应量（Cohen's d）：DPRT vs CS 差值除以合并标准差
- 论文「63.7%」的统计显著性未检验——二项检验可能不显著

**入门资源**：Wasserman, *All of Statistics* §3, §6, §8。

**自查标准**：从 JSON 加载 ratio 数据，用 scipy 计算 Spearman $\rho$ 和 Pearson r，跑 10000 次 bootstrap 得中位数 95% CI。

#### 6.M.12 详细说明：数值线性代数——浮点误差、条件数

**这是什么**：IEEE 754 双精度浮点数有约 16 位有效数字（机器精度 ε≈2.2×10⁻¹⁶）。条件数 κ(A) = ‖A‖·‖A⁻¹‖ 衡量矩阵求逆的数值稳定性：解的相对误差 ≤ κ × ε。

**为什么和 RadonShadow 相关**：论文报告 MUB↔DPRT 等价验证误差 < 10⁻¹⁴——这接近浮点精度（~10⁻¹⁶），表明等价性是「真」的而非浮点巧合。但如果反演矩阵条件数很大（如 d=2039 时），10⁻¹⁴ 的误差可能部分来自条件数放大。d=64 ratio=15.262 的灾难性失败极可能是条件数爆炸导致浮点误差淹没了真实信号。

**核心内容**：
- 双精度浮点：52 位尾数，约 15.95 位十进制有效数字
- 条件数 κ 决定「输入误差放大倍率」：输出误差 ≤ κ × 输入误差
- Ratio 计算的每一步（矩阵求逆、DFT、内积）都有数值误差累积
- 验证等价性的正确做法：比较两种方法各自重建的 ρ——如果两者本身都有浮点误差，差值 10⁻¹⁴ 可能低估真实差距
- d=64 的灾难：κ 爆炸 → ratio=15.262 本质上是噪声

**入门资源**：Trefethen & Bau, *Numerical Linear Algebra* §12–15。

**自查标准**：对 d=3 的 MUB 投影矩阵（12×9），手算条件数，估算反演误差理论上界。

#### 6.M.13 详细说明：量子态断层扫描基线方法（MLE、LI）

**这是什么**：Linear Inversion（LI）= 直接求解 tr(E_i $\rho$)=p_i 方程组反推 ρ。最大似然估计（MLE）= 最大化似然 L($\rho$)=Π p_i($\rho$)^{n_i}，约束 ρ≥0 和 tr($\rho$)=1，通常用迭代 RρR 算法求解。

**为什么和 RadonShadow 相关**：论文只对比了 Classical Shadow，没有对比 MLE 和 LI。但如果 CS 已知优于 MLE（在样本效率上），而 RadonShadow 优于 CS，那么传递性成立。然而论文没有引用这一对比——你需要自己判断传递性是否成立。中小维度（d≤30）中 MLE 仍然实用，RadonShadow 应在此范围内与 MLE 直接对比。

**核心内容**：
- LI：最简单但得到的 ρ̂ 可能非半正定 → 需要投影到物理态空间
- MLE：保证 ρ̂ 是合法量子态，但迭代收敛慢 → 大 d 不实用
- CS vs MLE：HKP 2020 证明 CS 在样本复杂度上指数优于 MLE
- RadonShadow vs MLE：没有直接对比 → 论文最明显的缺失基线
- 对 d≤30：MLE 仍实用，是 RadonShadow 最应补充的对比实验

**入门资源**：Paris & Řeháček (eds.), *Quantum State Estimation* §1–3。

**自查标准**：对 d=3，从论文的 MUB 投影数据分别实现 LI 和 MLE 重建 ρ，对比 CS 和 DPRT 的 ratio。

#### 6.M.14 详细说明：有限环 Z/p^k Z 上的调和分析

**这是什么**：当维度不是素数幂时，Z/p^k Z（整数模 p^k）是一个环而非域——非单位元没有乘法逆元。其上的调和分析比有限域复杂：加法特征标仍可用 e_{p^k}(x)=exp(2πix/p^k)，但乘法结构退化。Galois 环 GR(p^k, m) 是有限域的「提升」。

**为什么和 RadonShadow 相关**：论文合数维推广目前只用了 Local MUB Product（因子独立测量）。要从根本上推广 DPRT 到任意维 d=p^k，需要建立 Z/p^k Z 上的离散 Wigner 函数和 DPRT 对应。d=16 极好而 d=64 崩溃的现象，可能与 Z/2^k Z 中 2-adic 指数增长有关。

**核心内容**：
- Z/p^k Z 的零因子集合 = pZ/p^k Z，大小为 p^{k−1}——这些元素没有乘法逆
- 环上仍可定义 DFT，但频域不再有完美正交性（非零因子与零因子混合）
- Galois 环是有限域的「提升」，保留了很多域的性质
- DPRT 推广到 Z/p^k Z 的关键挑战：某些投影方向上的直线不再均匀覆盖像素
- d=16 vs d=64 的反差：2⁴ vs 2⁶ 的环结构差异可能解释了阈值效应

**入门资源**：Klappenecker & Rötteler 2005, "Mutually unbiased bases are complex projective 2-designs"。

**自查标准**：写出 Z/8Z 的零因子集合和单位群，对比 F_8 的乘法表，理解为什么 DPRT 在 Z/8Z 上某些投影方向信息丢失。

#### 6.M.15 详细说明：Mackey 诱导表示理论

**这是什么**：Mackey 理论将有限群 G 的表示为子群 H 的表示「诱导」而来。核心公式：Ind_H^G(V) 的基是陪集 G/H 上的截面，G 的作用通过陪集置换和 H 的内部作用组合给出。Frobenius 互反律是核心工具。

**为什么和 RadonShadow 相关**：这是博士路线 6（有限群确定性阴影框架）的理论基础。如果想「遍历全部 MUB」推广到「遍历群的全部不可约表示」，就需要诱导表示——MUB 可视为 Heisenberg-Weyl 群经 Mackey 诱导的特例。风险在于：推广到任意有限群后 DPRT-RadonShadow 对应是否还存在。

**核心内容**：
- 诱导表示 Ind_H^G(ρ_H) 的维度 = [G:H] × dim(ρ_H)
- MUB 基向量可视为 Heisenberg-Weyl 群在 $\mathbb{F}_p$ 上的诱导表示的自然基
- Frobenius 互反律：⟨Ind_H^G ρ_H, τ_G⟩_G = ⟨ρ_H, Res_H^G τ_G⟩_H
- 小群方法：将不可约表示按惯性子群的表示分层分类
- 博士路线 6 是高风险高回报：需要证明对任意有限群存在 DPRT 类比

**入门资源**：Serre, *Linear Representations of Finite Groups* §7。

**自查标准**：对 S_3（阶 6）和其 2 阶子群，能手算诱导表示的特征标表。

#### 6.P.1 详细说明：量子力学公理体系

**这是什么**：四条公理。(1) 孤立系统的态空间是 Hilbert 空间 H。(2) 时间演化由酉矩阵 U(t)=exp(−iHt/ħ) 描述。(3) 测量由一组 Hermitian 算符 {M_k} 决定，结果 k 的概率为 tr(M_k $\rho$ M_k†)。(4) 复合系统的态空间为 H_A⊗H_B。

**为什么和 RadonShadow 相关**：论文中「测量」一词同时出现在纯数学（§1.4 的投影测量）和物理语境（§4.4 噪声通道）中。理解公理 3 才能衔接——投影测量是 M_k = |e_k⟩⟨e_k| 的特例，而一般性 POVM 测量是 M_k 不必正交的推广。噪声通道（公理 2 的推广）也需要 Kraus 算符表示。

**核心内容**：
- 公理 2 的数学类比：线性 ODE 的解矩阵 exp(A)，其中 A 是反 Hermitian
- 公理 3 的数学类比：统计中「抽取样本 → 条件概率更新」
- POVM 是投影测量的推广（不需要算符正交），允许「软判决」
- 密度矩阵的数学定义（§1.2）恰好是公理 1+2+4 的推论
- 不需要理解 ħ（约化普朗克常数）——所有公式中它都被归一化掉了

**入门资源**：Nielsen & Chuang §2.1−2.2。

**自查标准**：从密度矩阵公理出发，证明 tr($\rho$)=1 在酉演化下保持不变，投影测量后仍为 1。

#### 6.P.2 详细说明：密度矩阵的物理诠释

**这是什么**：密度矩阵 $\rho$ 有两种解读。(1) 纯态 ρ=|ψ⟩⟨ψ| 来自对系统最大的知识：我们确切知道态是什么。(2) 混态 ρ=Σ p_i|ψ_i⟩⟨ψ_i| 来自经典概率混合：系统以概率 p_i 分别处于纯态 |ψ_i⟩。数学类比：纯态 = 单纯形的顶点，混态 = 单纯形内部的凸组合。

**为什么和 RadonShadow 相关**：论文的数值实验对纯态和混态都做了测试。理解「混态来自经典混合」（而非量子叠加）后，ratio 在不同态上的变化就有物理直觉——混态的 Wigner 函数更平滑（类似高斯模糊），DPRT 反演更稳定。

**核心内容**：
- 纯态 ↔ rank 1 ↔ ρ²=ρ（幂等）；混态 ↔ rank>1 ↔ ρ²≠ρ
- 密度矩阵的非对角元（ρ_{ij}, i≠j）编码量子相干性——这是经典概率没有的
- 混态混合：Σ p_i ρ_i 保持半正定和迹 1（概率单纯形的凸性）
- 最大混态 I/d：所有特征值相等 = 完全无知
- 退相干：非对角元衰减 → 混态化，这是量子→经典过渡的数学形式

**入门资源**：Nielsen & Chuang §2.4。

**自查标准**：给定两个纯态 |0⟩ 和 |+⟩=(|0⟩+|1⟩)/√2，对比 ρ₁=½|0⟩⟨0|+½|+⟩⟨+|（经典混合）和 |ψ⟩=(|0⟩+|+⟩)/N（量子叠加）的密度矩阵，指出关键差异。

#### 6.P.3 详细说明：量子噪声通道模型

**这是什么**：噪声通道是密度矩阵空间上的 CPTP（完全正定迹保持）映射。Kraus 表示：E($\rho$)=Σ_k E_k $\rho$ E_k† 且 Σ_k E_k† E_k = I。四种基本通道有明确的物理对应：去极化（环境随机扰动）、振幅阻尼（能量损失）、相位阻尼（相位扰动无能量损失）、比特翻转（经典错误）。

**为什么和 RadonShadow 相关**：论文 §4.4 测试了全部四种噪声通道，发现噪声有时反而增强 DPRT 优势。理解 Kraus 算符才能理解「噪声为什么有时是朋友」——中等噪声充当了 Wigner 函数的正则化器，抑制了高频成分导致的 DPRT 反演振荡。

**核心内容**：
- 去极化 E($\rho$) = (1−λ)ρ + λ·I/d：数学类比 → 向均匀分布做凸组合
- 振幅阻尼 E₀=diag(1,√(1−λ)), E₁=[[0,√λ],[0,0]]：只有基态是稳态
- 相位阻尼 E₀=diag(1,√(1−λ)), E₁=diag(0,√λ)：非对角元衰减速率 λ
- 比特翻转 E₀=√(1−λ)·I, E₁=√λ·X：以概率 λ 翻转 |0⟩↔|1⟩
- 噪声作为正则化器：Hilbert 空间类比 → L2 正则化抑制高频分量

**入门资源**：Nielsen & Chuang §8.1−8.3。

**自查标准**：对 d=2 去极化通道，手算 E(|0⟩⟨0|)，验证 λ=1 时退化为 I/2。

#### 6.P.4 详细说明：量子断层扫描的标准方法

**这是什么**：完整量子态断层扫描（QST）是重建未知 $\rho$ 的全部 d²−1 个独立参数的标准流程。需要至少 $d+1$ 组完备的测量基，测量 d(d+1) 个投影概率，然后通过最大似然估计这些独立参数。

**为什么和 RadonShadow 相关**：你的竞争基线是什么？HKP 的 Classical Shadow 赢了标准 QST（MLE），但 RadonShadow 只对比了 CS。论文中缺少 RadonShadow vs MLE 的直接对比——中小维度（d≤30）中 MLE 仍实用，这是论文最明显的缺失基线。你需要自己补上这个对比才能判断 RadonShadow 的真正地位。

**核心内容**：
- 标准 QST 需要 d²−1 个测量基 → d³ 次投影（远多于 RadonShadow 的 (p+1)×p）
- CS 的样本复杂度 $O(\log d) 指数优于 QST 的 d²
- RadonShadow 继承了 CS 的样本效率但消除了随机方差
- 中小维：MLE 可行 → RadonShadow vs MLE 对比应优先完成
- 大维：MLE 不可行 → CS 是唯一实用基线

**入门资源**：Paris & Řeháček (eds.), *Quantum State Estimation* §1–3；HKP 2020 的对比讨论。

**自查标准**：对 d=3，从 MUB 投影概率实现 LI 和 MLE 重建，与论文的 CS 和 DPRT 结果对比 ratio。

#### 6.P.5 详细说明：量子纠缠与可分性判据

**这是什么**：纠缠是指两体态 ρ_{AB} 不能写为 Σ p_i ρ_A^{(i)}⊗ρ_B^{(i)}（可分态）。数学类比：矩阵的秩——可分的密度矩阵 = 可分解的「秩 1」张量积的凸组合，纠缠的密度矩阵 = 不可分解。PPT（部分转置）判据：若 ρ_{AB}^{T_A} 有负特征值 → 纠缠。

**为什么和 RadonShadow 相关**：合数维 Local MUB Product 方法假设「每个子系统独立测量后做张量积」——这等价于假设态是可分的。如果子系统之间有纠缠（如 Bell 态），Local Product 测量的信息内容少于全局 MUB 测量——DPRT ratio 会受影响。你不能在没有纠缠知识的情况下判断合数维结论的适用范围。

**核心内容**：
- 可分态：Σ p_i · ρ_A^{(i)}⊗ρ_B^{(i)}（经典混合 + 张量积结构）
- PPT 判据：ρ^{T_A} 的非负性是可分性的必要条件（2⊗2 和 2⊗3 也是充分的）
- Local MUB Product 对纠缠态的偏迹引入额外不确定性
- 论文中合数维结果隐含假设「态可分或近似可分」
- 对于高度纠缠的态，应使用全局 MUB 而非 Local Product

**入门资源**：Nielsen & Chuang §2.5（张量积）+ §12.5（可分性判据）；Horodecki 1996 PPT 判据原始论文。

**自查标准**：对 Bell 态 (|00⟩+|11⟩)/√2，手算 $\rho$ 和 ρ^{T_A} 的特征值，判定 PPT 判据是否检测到纠缠。

#### 6.P.6 详细说明：超导量子比特 / 离子阱硬件概览

**这是什么**：超导量子比特使用 Josephson 结的非线性电感和微波脉冲实现二能级系统（|0⟩,|1⟩）。离子阱使用激光冷却和 Paul 阱囚禁的单个离子电子能级。两者目前都是「标准 qubit」——高维 generalization 需要 qutrit（三能级系统）。

**为什么和 RadonShadow 相关**：实验路线（路线 2）要求在实际量子硬件上实现 d=3 qutrit 的全 MUB 测量。但标准超导 qubit 和离子阱 qubit 都设计为二能级系统——实现 qutrit 需要利用更高能级（|2⟩态），这会引入能级泄漏和额外退相干。理解硬件约束后才能判断路线 2 的可行性时间线。

**核心内容**：
- 超导 qubit：Josephson 能量 E_J/充电能 E_C 比决定了非简谐性 → 泄漏到 |2⟩ 的速率
- 离子阱：长寿命的亚稳态可充当 |2⟩ 态，相干时间 > 1s
- qutrit 测量：需要在 3 个能级间进行投影 → 需要至少 2 个频率的脉冲
- MUB 测量在硬件上的额外开销：方向切换需要重新校准
- 离子阱在 qutrit 上比超导更成熟（天然的多能级结构）

**入门资源**：Nielsen & Chuang §7（量子计算机物理实现）；Google Quantum AI 和 IonQ 的公开技术文档。

**自查标准**：能回答「为什么 d=3 的全 MUB 测量在超导芯片上比在离子阱上更难」。

#### 6.X.4 详细说明：真效应 vs 伪影判别

**这是什么**：伪影（artifact）是实验设计中引入的、非客观存在的信号。常见来源：(1) 随机种子的偶然有利性 (2) 浮点误差的系统偏差 (3) 单一样本不代表统计总体。判别方法：多种子验证 + bootstrap 置信区间 + 交叉验证。

**核心内容**：
- 换种子法：用 10 个不同 random seed 重跑 d=17 2-local → 观察 ratio 分布
- Bootstrap：对已有数据重抽样 10000 次 → 得 95% CI
- 交叉验证：一半数据训练、一半验证 → 看是否过拟合于特定随机态
- 控制实验：只改一个变量（如原根），其余不变 → 量化效应大小
- 立即动手：

## 第 8 章：术语对照表——量子物理名 → 纯数学定义

| 量子物理术语 | 纯数学定义 |
|:--|:--|
| **量子态** | 一个 $d \times d$ Hermitian 矩阵 ρ，半正定，tr($\rho$) = 1 |
| **纯态** | $\rho$ = \|ψ⟩⟨ψ\|，秩 1，其中 \|ψ⟩ 是单位复向量 |
| **混态** | $\rho$ 的秩 > 1，不是任何纯态的外积 |
| **密度矩阵** | = 量子态（同义词） |
| **可观测量** | 一个 $d \times d$ Hermitian 矩阵 O |
| **期望值** | tr(Oρ)，矩阵的 Frobenius 内积的推广 |
| **测量** | 给定正交基 B，提取 $\rho$ 的对角元素 ⟨e_i\|ρ\|e_i⟩（即概率分布 p_i） |
| **测量基** | 一个正交基 B = {\|e₁⟩, …, \|e_d⟩} |
| **MUB（互无偏基）** | 满足 \|⟨e₁_i\|e₂_j⟩\| = $1/\sqrt{d}$ 的两组正交基 |
| **Wigner 函数** | 一个 $p \times p$ 网格上的实值函数，是量子态在相空间中的表示 |
| **投影算子** | 矩阵 Π = \|e⟩⟨e\|，其中 \|e⟩ 是单位向量 |
| **测量信道** | 线性映射 M: $\rho$ ↦ M($\rho$)，描述测量对量子态的平均效果 |
| **影子快照（Snapshot）** | 单次测量后构造的「态估计」ρ̂_snap = (d+1)\|e⟩⟨e\| − I |
| **影子范数** | \|\|O\|\|²_shadow = d·tr(O²)（对 MUB 阴影），决定估计 O 所需的最小测量次数 |
| **k-local 可观测量** | 只涉及 k 个子系统的可观测量（如 1-local $\approx$ 单个矩阵元的近对角结构） |
| **去极化噪声** | $\rho$ ↦ (1−λ)ρ + λ·I/d，把态向最大混态「洗」 |
| **Shot noise** | 有限次测量得到的频率 $\neq$ 真实概率导致的统计误差 |
